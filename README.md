# Titanic Mini Visualization Dashboard – Data Science Internship Task 4

## 📌 Project Overview
This project is developed as part of the **Data Science with Python Internship**.  
The objective of this task is to build a **Mini Data Visualization Dashboard** using Python libraries to analyze the Titanic dataset and present insights using multiple charts.

The dashboard helps in understanding survival patterns based on gender, passenger class, and age distribution.

---

## 🎯 Objective
The main goal of this task is to:
- Perform **data cleaning**
- Apply **basic feature engineering**
- Create **multiple visualizations**
- Communicate insights using graphs

---

## 📂 Dataset
The dataset used in this project is the **Titanic dataset**, which contains passenger information such as:

- Passenger class
- Gender
- Age
- Fare
- Family members
- Survival status

The dataset is loaded using the **Seaborn library**.

---

## 🛠 Tools & Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab / Jupyter Notebook
- GitHub

---

## 📊 Data Processing Steps

1. **Dataset Loading**
   - Titanic dataset loaded using Seaborn.

2. **Data Cleaning**
   - Missing values in the Age column were filled using median.
   - Missing values in the Embarked column were filled using mode.

3. **Feature Engineering**
   - Created Age Groups
   - Created Family Size feature

4. **Exploratory Data Analysis**
   - Survival count analysis
   - Gender-wise survival comparison
   - Passenger class survival comparison
   - Age distribution analysis

---

## 📈 Visualizations Created

The dashboard includes the following charts:

1️⃣ Survival Count Bar Chart  
2️⃣ Survival Rate by Gender  
3️⃣ Survival Rate by Passenger Class  
4️⃣ Age Distribution Histogram  
5️⃣ Correlation Heatmap  

These visualizations help in understanding patterns and relationships within the dataset.

---

## 🔍 Key Insights

- Female passengers had a significantly higher survival rate compared to male passengers.
- Passengers traveling in **first class** had better survival chances.
- The majority of passengers were between **20 to 40 years old**.
- Passenger class shows a strong relationship with survival probability.
